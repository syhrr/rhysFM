# rhysFM
rhysFM is my first ever proper project. 
###  Why?
- I wanted to visualise all of my LastFM scrobble data from the moment I started using it to now.
- Needed a way to do this so I chose to link it with machine learning.

## Goals
rhysFM should do the following:
- Discover distinct eras in a listener's taste
- Show how their taste evolved over time
- Do it with an actual ML pipeline using clustering
- Make it useable by anyone via a web app that takes a public Last.fm username.
